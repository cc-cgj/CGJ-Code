### 一般定时任务都是用cron表达式去表示时间的

```Cron
  * * * * * *：每秒执行一次任务。
  0 * * * * *：每分钟的整点执行一次任务。
  0 0 * * * *：每小时的整点执行一次任务。
  0 0 * * * *：每天的午夜执行一次任务。
  0 0 * * 1 *：每周一的午夜执行一次任务。
  0 0 1 * * *：每月的1号午夜执行一次任务。
  0 0 1 1 * *：每年的1月1日午夜执行一次任务。

### Cron表达式的典型格式如下：
  *    *    *    *    *    *   *(否)
  ┬    ┬    ┬    ┬    ┬    ┬   ┬  
  │    │    │    │    │    │   │
  │    │    │    │    │    │   └── 年份（1970 - 2099）
  │    │    │    │    │    └── 星期（0 - 6，0表示星期日）
  │    │    │    │    └───── 月份（1 - 12）
  │    │    │    └────────── 日（1 - 31）
  │    │    └─────────────── 小时（0 - 23）
  │    └──────────────────── 分钟（0 - 59）
  └───────────────────────── 秒（0 - 59）
