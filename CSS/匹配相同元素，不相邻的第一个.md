### 1. 巧用 :has() 与后代选择器 ~ 和 :not() 组合

```css
.box .cc:not(.cc ~ .cc) {
  color: red;
  font-size: 24px;
}
```

### 2.使用 :nth-child 伪类的现代特性

从 Chrome 111，:nth-child() 提供了一种更为强大的特性，让我们能够方便的实现上述的效果，这种语法就是 :nth-child(1 of .foo)。

```css
.box .cc:nth-child(1 of .cc) {
  color: red;
  font-size: 24px;
}
```
